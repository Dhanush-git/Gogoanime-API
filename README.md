# API Documentation

*Running on localhost*

**GET Popular Anime**
```
http://127.0.0.1:3000/Popular
```

**GET New Seasons**
```
http://127.0.0.1:3000/NewSeasons
```


**GET Anime**
```
http://127.0.0.1:3000/getAnime/"anime_id"
```


**GET Anime Episode**
```
http://127.0.0.1:3000/getEpisode/"episode_id"
```
